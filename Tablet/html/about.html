About Tablet Skin

This skin was designed for use with web tablets, such as the Apple iPad.

It was designed in June/July 2010 by SJB in Auckland, New Zealand, and was based on the Nokia 770 skin.

It uses some icons from Travis' free IconPack.

Nokia770 skin was created by Ben Klaas, December 2005, Minneapolis.

The Touch skin was the base code used for creating Nokia770. Touch was created by Stefan Christiansson, February 2004, Stockholm.
